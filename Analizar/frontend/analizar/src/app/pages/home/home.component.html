FAQ

1. ¿La aplicación tiene algún costo?

No, la aplicación es completamente gratuita. No se requiere ningún tipo de pago para acceder a sus funcionalidades.

2. ¿Es necesario tener conocimientos previos para utilizar la aplicación?

Aunque la aplicación es intuitiva, se recomienda seguir el tutorial adjunto para aprovechar al máximo todas sus características. No se necesitan conocimientos previos para comenzar a utilizarla.

3. ¿AnalizAr vende medidores eléctricos inteligentes?

Sí, AnalizAr ofrece medidores eléctricos inteligentes como parte de sus servicios. La compra de estos dispositivos no es obligatoria, pero todos nuestros equipos vienen con una garantía escrita de 12 meses para cubrir cualquier defecto. Para obtener más información, comunicate con nuestros agentes comerciales.

4. ¿Puedo usar AnalizAr en Android?

¡Sí! AnalizAr está disponible para dispositivos Android. Podés descargar la APK directamente desde nuestro sitio web. Solo tenés que hacer clic en el botón que indica "Descargar App" y seguir las instrucciones para instalarla en tu dispositivo. ¡Disfrutá de todas las funcionalidades de AnalizAr en tu móvil!

5. ¿La aplicación está disponible en varios idiomas?

Actualmente, la aplicación está disponible en español. Sin embargo, estamos trabajando constantemente para expandir nuestro soporte de idiomas y mejorar la accesibilidad para nuestros usuarios.
